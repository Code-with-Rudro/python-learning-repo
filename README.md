<h4>cmmond lines</h4>
<br>
git config --global user.name “[firstname lastname]”
<em>
set a name that is identifiable for credit when review version history </em>
<br>
git config --global user.email “[valid-email]”
<em>
set an email address that will be associated with each history marker </em>
<br>
git config --global color.ui auto<em>
set automatic command line coloring for Git for easy reviewing</em>
<br>
git init<em>
initialize an existing directory as a Git repository</em>
<br>
git clone [url]<em>
retrieve an entire repository from a hosted location via URL</em>
<br>
git add <--file name-->
<em>to add files </em>
<br>
git commit -m "comment"
<em> to pass a message of action</em>
<br>
git push origin main 
<em>to push local cnt to remote repo</em>
